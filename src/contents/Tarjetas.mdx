# **Formulario online: tarjetas de credito**
<br />
En esta ocasión, trabaje para la creación de un formulario en línea para solicitar una tarjeta de crédito.
<br />
## Tecnologías Utilizadas
- HTML 
- CSS
- SQL
- JavaScript
<br />
## Descripción
La creación del código en si fue bastante corto, la complicación estuvo en el análisis y diseño del mismo, pues tenía que recopilar los datos de la forma más eficiente y ser lo más claro posible para evitar errores de "capa 8".
<br/>
Mi función principal fue en el diseño de la base de datos para que fuese lo más óptima posible, a la vez que aportaba con soluciones en el análisis para el formulario en general, sobre qué datos pedir y cuales eran innecesarios.
