# **Desarrollo Web: marketplace**

<br />
<br />
<br />

Se trataba de la creación de un **MARKETPLACE** como proyecto principal en la clase de ingeniería de software, siendo que fue un trabajo en equipos con la finalidad de desarrollar a un nivel parecido al mundo laboral.
<br />
## Tecnologías Utilizadas
- HTML
- CSS
- SQL
- Node.js
- React

<br />
## Descripción
Este proyecto consistió en el diseño y desarrollo de una web de ventas en internet, en donde los usuarios pudiese registrarse, buscar productos de venta o publicar sus propios artículos para su venta, la web no se encargaba de la gestión de ventas, su función era dar un espacio para que las personas pudieran contactarse para la venta y compra de artículos.
<br />
En el proceso, se tuvieron que cumplir con varios requerimientos que el cliente (el catedrático) designo desde el comienzo, tales como, organización por categorías, la posibilidad de crear chats privados o públicos, subir fotos y videos, gestión de denuncias, entre otros.

<br />
En este proyecto me desempeñe como gestor de bases de datos y colabore en l creación del front-end de la web, dando soluciones simples pero eficientes a los retos que se presentaban en el desarrollo.
